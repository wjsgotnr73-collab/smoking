# 비흡연자와 흡연자 모두를 위한 흡연 장소 찾아 주는 프로그램 


개요

길거리 담배 꽁초와 흡연으로 인해 비흡연자의 간접 흡연 피해로 흡연자와 비흡연자간의 마찰이 줄어들지 않고 있지만, 
정작 흡연 구역이 마땅히 없어 골목길이나 다른 사람들이 담배를 많이 피운 장소에 가서 흡연을 하는 등의 상황이 존재

해결책

공공데이터 포털의 지자체의 흡연 구역 데이터를 파이썬으로 부산 지역의 흡연 구역 위치를 알려 주는 터미널 기반의 정보 제공 시스템을 개발

주요 타겟: 정보 부족으로 흡연장을 찾지 못하는 부산 지역 방문객, 이외 흡연자들

기대효과

비흡연자와의 사회적 갈등 문제 감소
파편화된 공공데이터를 하나로 모아 지역 데이터가 추가될 때마다 흡연 장소를 쉽게 확인할 수 있는 통합 안내 프로그램으로 발전 시킬 수 있다
